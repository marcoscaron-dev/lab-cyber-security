# 🔐 Cyber Security Lab

Hands-on cybersecurity lab focused on cloud security, detection, and threat analysis.

## 📌 Objectives
- Simulate real-world security scenarios
- Practice detection and response
- Explore cloud security (AWS/Azure)

## 🛠️ Technologies
- Python
- AWS (IAM, S3, CloudTrail)
- Linux
- SIEM concepts

## 🔍 Projects

### Cloud Security Checks (AWS)
Scripts to identify:
- Public S3 buckets
- Over-permissive IAM roles
- Basic misconfigurations

### Endpoint Monitoring Simulation
- Simulated suspicious activity detection
- Basic alert generation

## 🎯 What this demonstrates
- Cloud security fundamentals
- Detection mindset
- Security automation skills
